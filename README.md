# 🎯 客服智慧抽籤與輪值系統 (CS Lottery & Rotation Tool)

專為客服團隊設計的高質感、多功能抽籤與輪值管理工具。本工具完全基於瀏覽器運行，使用本地儲存（LocalStorage），無須後端伺服器，且支援離線使用。

---

## 🌟 核心特色

1. **🎨 頂級視覺美學 (Premium Glassmorphism UI)**
   - 採用現代磨砂玻璃質感設計與流暢微動畫。
   - 支援「深色模式」與「淺色模式」一鍵切換，配合日夜間客服值班視覺需求。

2. **🎰 擬真機械老虎機抽籤 (Realistic Slot Machine)**
   - **實體拉桿動效**：支援點擊右側的立體紅色球體拉桿，觸發物理拉下拉桿並回彈的動畫。
   - **3D 圓柱滾輪遮罩**：具備立體光影遮罩，讓名字滾動呈現如真實滾筒旋轉時的立體弧形與陰影。
   - **卡位回彈物理動效**：採用自訂貝氏曲線減速，滾輪在停止時會稍微滾動超越紅線，隨即「微微倒彈」精準卡住，還原實體滾輪齒輪卡位感。

3. **🎬 抽籤過程自動影音錄影 (Auto Screen Recording)**
   - 開啟錄影開關後，啟動抽籤會自動請求瀏覽器「分頁錄影」權限。
   - 自適應錄製老虎機滾動與中籤慶祝畫面。
   - 關閉慶祝視窗時，會自動結束錄影並下載高畫質 `.webm` 格式影片作為存證，無需額外安裝任何軟體。

4. **⚙️ 完整的名單管理與在線主題編輯**
   - **在線標題編輯**：直接雙擊頂部的主題標題即可線上即時修改，系統會即時寫入本地儲存，無須頻繁點按確認。
   - **批次匯入**：直接複製貼上人名名單即可完成匯入，免除逐一輸入的繁瑣。
   - **啟用狀態切換**：客服同仁請假時，可取消勾選暫時停用，該員便不會參與抽籤，銷假後一鍵重新啟用，免去刪除後重新輸入的麻煩。
   - **權重調整 (0.1 至 10)**：可調整特定人員的被抽中機率（例如：組長機率減半、新人機率加倍）。

5. **🔄 防重複抽籤規則 (不重複抽取)**
   - 開啟「不重複抽籤（抽出後排除）」後，抽中的人員會自動被移出隨機池，直到手動重置或所有人都被抽過一輪（抽完後系統會自動循環重置），非常適合分派值日生或輪班工作。

6. **📜 本地歷史紀錄與 CSV 導出**
   - 自動記錄每次抽籤的主題、時間、模式、抽中人選以及當時的候選池狀態。
   - 提供「匯出 CSV」功能，且檔案特別注入 UTF-8 BOM 標頭，確保使用 Microsoft Excel 開啟時中文姓名不亂碼。


## 🛠️ 安裝與啟動方式

1. **直接啟動（推薦）**：
   - 由於此系統為純前端靜態應用程式，您可以直接在瀏覽器中開啟 `index.html` 即可開始使用！

2. **使用開發伺服器運行**：
   - 如果您想在本地運行 Web Server：
     ```bash
     # 您可以在專案目錄下啟動一個簡易伺服器，例如使用 python:
     python -m http.server 8000
     # 或者使用 Nodejs 的 npx http-server:
     npx http-server .
     ```
   - 在瀏覽器中輸入 `http://localhost:8000` 即可訪問。

## 📖 客服常用情境範例

### 情境 A：分派每日值日生（不重複抽籤 + 自動錄影存證）
1. 在主題管理中新增「每日值日生」。
2. 批次匯入所有客服人員名單。
3. 勾選開啟「**不重複抽籤 (抽出後排除)**」及「**🎬 抽籤過程自動錄影**」。
4. 每天上班時，點選拉桿，點選瀏覽器跳出的提示「共享此分頁」，開始自動錄製。
5. 老虎機轉動並顯示中籤人，點選「確認並收起」後，系統會下載包含老虎機旋轉與中籤彈框的錄影影片，供群組發佈與存證。

### 情境 B：客服組長請假調整
1. 客服專員「阿強」請年假。
2. 進入「設定與管理」頁面，在名單中取消勾選「阿強」的啟用狀態。
3. 抽籤池中「阿強」的機率會立即變為 `0%`，其餘人員機率自動重新加權計算。
4. 進行抽籤時，絕對不會抽中「阿強」。
5. 阿強假期結束上班後，回到設定頁面重新勾選啟用即可！
