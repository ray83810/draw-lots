# 🎯 客服智慧抽籤與輪值系統 (CS Lottery & Rotation Tool)

專為客服團隊設計的高質感、多功能抽籤與輪值管理工具。本工具完全基於瀏覽器運行，使用本地儲存（LocalStorage），無須後端伺服器，且支援離線使用。

## 🌟 核心特色

1. **🎨 頂級視覺美學 (Premium Glassmorphism UI)**
   - 採用現代磨砂玻璃質感設計與流暢微動畫。
   - 支援「深色模式」與「淺色模式」一鍵切換，配合日夜間客服值班視覺需求。
   
2. **🎰 三大抽籤動畫模式**
   - **滾動老虎機**：適合單人或多人中籤的華麗滾動效果。
   - **幸運大轉盤**：經典的轉盤視覺，依據權重比例繪製，適合轉盤互動。
   - **互動翻卡牌**：將名單打散並隱藏在神秘卡牌中，由客服同仁親自點擊翻開，增加參與感與趣味性。

3. **⚙️ 完整的名單管理**
   - **主題切換**：可儲存多組範本（如：班表、值日生、特殊任務專員、午餐選擇）。
   - **批次匯入**：直接複製貼上人名名單即可完成匯入，免除逐一輸入的繁瑣。
   - **啟用狀態切換**：客服同仁請假時，可點擊「勾選框」暫時停用，該員便不會參與抽籤，銷假後一鍵重新啟用，免去刪除後重新輸入的麻煩。
   - **權重調整 (0.1 至 10)**：可調整特定人員的被抽中機率（例如：組長機率減半、新人機率加倍）。

4. **🔄 防重複抽籤規則 (不重複抽取)**
   - 開啟「不重複抽籤（抽出後排除）」後，抽中的人員會自動被移出隨機池，直到手動重置或所有人都被抽過一輪（抽完後系統會自動循環重置），非常適合分派值日生或輪班工作。

5. **📜 本地歷史紀錄與導出**
   - 自動記錄每次抽籤的主題、時間、模式、抽中人選以及當時的候選池狀態。
   - 支援關鍵字搜尋與主題篩選。
   - 提供「匯出 CSV」與「匯出 JSON」功能。CSV 檔案特別注入 UTF-8 BOM 標頭，確保使用 Microsoft Excel 開啟時中文姓名不亂碼。

## 🛠️ 安裝與啟動方式

1. **直接啟動（推薦）**：
   - 由於此系統為純前端靜態應用程式，您可以直接在瀏覽器中開啟 `index.html` 即可開始使用！

2. **使用開發伺服器運行**：
   - 如果您想在本地運行 Web Server：
     ```bash
     # 您可以在專案目錄下啟動一個簡易伺服器，例如使用 python:
     python -m http.server 8000
     # 或者使用 Nodejs 的 npx http-server:
     npx http-server .
     ```
   - 在瀏覽器中輸入 `http://localhost:8000` 即可訪問。

## 📖 客服常用情境範例

### 情境 A：分派每日值日生（不重複抽籤）
1. 在主題管理中新增「每日值日生」。
2. 批次匯入所有客服人員名單。
3. 勾選開啟「**不重複抽籤 (抽出後排除)**」。
4. 每天上班時，選擇「一次抽取 1 人」並點選開始抽籤。
5. 抽中者會自動被排除在剩餘抽籤池之外，下方進度條會顯示「抽籤池剩餘：5/6」。
6. 週五時，最後一位客服人員被抽中，下週一再次抽籤時，抽籤池會自動循環重置，開始新一輪的輪值！

### 情境 B：客服組長請假調整
1. 客服專員「阿強」請年假。
2. 進入「設定與管理」頁面，在名單中取消勾選「阿強」的啟用狀態。
3. 抽籤池中「阿強」的機率會立即變為 `0%`，其餘人員機率自動重新加權計算。
4. 進行抽籤時，絕對不會抽中「阿強」。
5. 阿強假期結束上班後，回到設定頁面重新勾選啟用即可！
